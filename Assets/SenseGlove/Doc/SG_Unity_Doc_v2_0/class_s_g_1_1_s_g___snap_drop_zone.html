Detailed Description

A DropZone that snaps a Grabable to a specific SnapPoint.

Member Enumeration Documentation

◆ SnapMethod

enum SG.SG_SnapDropZone.SnapMethod
strong

The way in which a SnapZone attaches objects to itself.

Enumerator
ObjectDependent 

The snapzone chooses which option to use, based on the Grabable's GrabType propery

Parent 

The Grabable becomes a child object of the dropzone. If it posesses a Rigidbody, it is marked as kinematic.

FixedJoint 

The DropZone creates a PhysicsJoint connection between this dropzone and the grabable rigidBody.
