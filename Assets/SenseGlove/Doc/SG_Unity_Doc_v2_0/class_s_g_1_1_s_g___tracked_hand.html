Member Enumeration Documentation

◆ TrackingMethod

enum SG.SG_TrackedHand.TrackingMethod
strong

The way this TrackedHand follows its TargetPosition

Enumerator
Default 

The hand matches the trackedObject's position and rotations, with offsets.

PhysicsBased 

The hand gets a rigidbody, which attempts to reach its targetRotation and -position

Disabled 

This script does not handle any tracking. Use this when making the hand a child of your trackedObject.
