◆ GrabType

enum SG.GrabType
strong

The way in which this Grabscript picks up SG_Interactable objects.

Enumerator
Follow 

The grabbed object's transform follows that of the GrabReference through world coordinates. Does not interfere with VRTK scripts.

FixedJoint 

A FixedJoint is created between the grabbed object and the GrabReference, which stops it from passing through rigidbodies.

Parent 

The object becomes a child of the Grabreference. Its original parent is restored upon release.
